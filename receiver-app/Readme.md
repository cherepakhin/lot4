Приложение для создания push-уведомления
==

| Название|Значение по умолчанию|Описание|
| --- |------- | ------ |
| KAFKA_HOST |127.0.0.1:9092| Хост Kafka |
| KAFKA_TOPIC |lot4-messages| Топик Kafka для входящих сообщений |
| PG_URL |jdbc:postgresql://localhost:5432/db| Хост PostgreSQL|
| PG_USER |postgres| Имя пользователя PostgreSQL |
| PG_PASSWORD |postgres| Пароль PostgreSQL |
